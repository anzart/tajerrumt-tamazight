# Tajerrumt n Tmaziɣt — Amazigh (Kabyle) Grammar Reference

An interactive, single-page web version of a reference manual on Amazigh (Berber)
grammar, with a focus on **Kabyle (Taqbaylit)**. It covers the alphabet,
phonological rules, punctuation, assimilation, the noun and the state of
annexation, the verb, pronouns, sentence structure, text types, numbers, the
Tifinagh script, a glossary, a bibliography, and an interactive quiz.

🌍 **Live site:** https://anzart.github.io/tajerrumt-tamazight/

## About the content

The content of this site is inspired by the **"Manuel de Référence de la
Grammaire Amazighe"** by **Juba Tala**:

👉 https://adrar-inu.blogspot.com/2025/11/manuel-de-reference-de-la-grammaire.html

This is a community re-presentation and it **may contain mistakes** — typos,
transcription errors, or misinterpretations introduced while adapting the
original manual into web form. If you spot something wrong, please help improve it!
